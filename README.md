### Pythonによる因数分解問題生成ツール<br>
## 概要
これは**pythonを用いることにより因数分解の問題を簡単に生成**し，さらに**LaTeXコードに変換**するプロジェクトです．<br>
元来因数分解の問題は適当な数を考えそれを展開することにより作成可能ですが，人力で実行するには大変面倒であります．そのため，pythonを用いることにより，[-9,9]の範囲で整数を2つ生成し，simpyを用いて展開．その後，自作したLaTeX変換関数を用いファイルへ書き出します．<br>
## 必要環境
LaTeXとPythonを用います．Pythonにてsympyをimportするために，それをinstallしておく必要があります．なお，LaTeXについては，大量にファイルをあげるとTimeoutしてしまう可能性があるため，overleafは推奨せず，VS codeなどほかの媒体を推奨します．
## 使い方
Pythonによるコードを実行し，出力されたファイルをLaTeX上で\input{}を用いてコンパイルし，PDFを得ます．なお，掲出したPythonコード内では整数の生成は(-9,9)の整数の範囲で，ファイルは5個出力されるようになっています．ここは任意に対応しているため，必要に応じて書き換えてください．
## 既知の制限事項
このプロジェクトでは，(x + 0)(x + 1)や(x + 3)(x + 3)など不自然な記法は排除していますが，(x + 2)(x + 7)と(x + 7)(x + 2)が同じファイル内に出てくる可能性を排除していません．また，完全にランダムに整数を生成しているために，難易度は不規則的であります．
